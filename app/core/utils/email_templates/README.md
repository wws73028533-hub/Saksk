# 邮件模板系统

## 概述

现代化的邮件模板系统，采用iOS 18风格设计，提供优雅、专业的邮件界面。

## 设计特点

- **iOS 18风格**：简约、现代的Apple美学
- **响应式设计**：完美适配桌面和移动设备
- **模块化结构**：易于维护和扩展
- **类型安全**：使用类型提示确保代码质量

## 模板结构

```
email_templates/
├── __init__.py          # 模板渲染引擎
├── base.html            # 基础模板（包含通用样式和布局）
├── bind_code.html       # 邮箱绑定验证码模板
├── login_code.html      # 登录验证码模板
└── reset_password.html  # 密码重置验证码模板
```

## 使用方法

模板系统已集成到 `EmailService` 中，无需额外配置。发送邮件时会自动使用新的模板。

## 添加新模板

1. 在 `email_templates/` 目录下创建新的HTML模板文件
2. 继承 `base.html` 模板
3. 在 `__init__.py` 的 `get_email_subject()` 函数中添加对应的主题
4. 在 `email_service.py` 的 `_render_email_template()` 中添加模板类型映射

## 设计规范

- **颜色方案**：
  - 绑定邮箱：绿色 (#34c759)
  - 登录验证：蓝色 (#007aff)
  - 密码重置：红色 (#ff3b30)

- **字体**：使用系统字体栈（San Francisco优先）
- **间距**：遵循8px网格系统
- **圆角**：16-20px圆角，营造现代感
- **阴影**：轻微阴影，增加层次感

## 自定义样式

可以通过覆盖 `base.html` 中的CSS变量来自定义样式，或直接在子模板中使用 `{% block extra_styles %}` 添加额外样式。
