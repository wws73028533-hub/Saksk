# TabBar图标说明

请在此目录下放置以下图标文件：

1. home.png / home-active.png - 首页图标（未选中/选中）
2. practice.png / practice-active.png - 练习图标（未选中/选中）
3. mine.png / mine-active.png - 我的图标（未选中/选中）

图标规格建议：
- 尺寸：81px × 81px
- 格式：PNG
- 未选中状态：灰色 (#7A7E83)
- 选中状态：蓝色 (#007AFF)

可以使用图标库如：
- IconFont
- Ionicons
- Material Icons
