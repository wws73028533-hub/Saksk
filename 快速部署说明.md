# 快速部署说明

## 1. 设置环境变量

### Windows (PowerShell)
```powershell
$env:FLASK_ENV="production"
$env:SECRET_KEY="your-secret-key-here"
```

### Linux/Mac
```bash
export FLASK_ENV=production
export SECRET_KEY="your-secret-key-here"
```

或创建 `.env` 文件：
```bash
FLASK_ENV=production
SECRET_KEY=your-secret-key-here
```

## 2. 生成密钥

```bash
python -c "import secrets; print(secrets.token_urlsafe(32))"
```

## 3. 安装依赖

```bash
pip install -r requirements.txt
```

## 4. 启动方式

### 方式一：直接运行（开发/测试）
```bash
python run.py
```

### 方式二：使用 Gunicorn（推荐生产环境）
```bash
# 基本启动
gunicorn -w 4 -b 0.0.0.0:5000 run:app

# 使用配置文件
gunicorn -c gunicorn_config.py run:app
```

### 方式三：使用启动脚本
```bash
# Linux
chmod +x start_production.sh
./start_production.sh

# Windows
start_production.bat
```

## 5. 验证部署

访问 `http://your-server-ip:5000` 检查应用是否正常运行。

## 6. 使用 Nginx 反向代理（可选）

参考 `生产环境部署指南.md` 中的 Nginx 配置。

## 重要提示

1. **必须设置 SECRET_KEY**：生产环境必须设置强随机密钥
2. **禁用 DEBUG**：生产环境自动禁用 DEBUG 模式
3. **使用 Gunicorn**：生产环境建议使用 Gunicorn 而不是 Flask 内置服务器
4. **配置 HTTPS**：生产环境建议使用 HTTPS
5. **定期备份**：定期备份数据库文件 `instance/submissions.db`
